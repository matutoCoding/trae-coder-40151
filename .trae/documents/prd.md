## 1. 产品概述
口腔诊所牙椅排队调度移动App，解决诊所患者候诊混乱、牙椅资源冲突、VIP/急诊优先等问题。面向诊所前台、医生和患者，实现智能排队叫号、优先插队、牙椅排期和冲突校验一体化管理。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 前台/管理员 | 管理员分配账号 | 取号排队、叫号、插队、牙椅建档、排期管理、退订 |
| 医生 | 管理员分配账号 | 查看队列、叫号、标记诊疗完成、复诊预约 |
| 患者 | 现场取号/线上预约 | 取号、查看排队位置、预约、退订 |

### 2.2 功能模块
1. **排队叫号页面**：取号登记、实时队列展示、叫号播报、当前就诊信息
2. **优先插队页面**：VIP/急诊标识、插队操作、优先级排序展示
3. **牙椅排期页面**：牙椅状态总览、时段排班表、预约占用可视化
4. **冲突校验页面**：预约冲突提示、退订释放、复诊衔接

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 排队叫号 | 取号登记 | 患者输入姓名、手机号、就诊类型（普通/VIP/急诊）取号，生成排队序号 |
| 排队叫号 | 实时队列 | 展示当前等待队列，显示序号、姓名、类型标签、等待时长 |
| 排队叫号 | 叫号操作 | 医生/前台点击叫号，队列首位患者上椅，播报提示音 |
| 排队叫号 | 就诊状态 | 显示当前各牙椅正在就诊的患者信息及开始时间 |
| 优先插队 | VIP/急诊标识 | 患者可标记为VIP或急诊，队列中显示醒目标签 |
| 优先插队 | 插队操作 | 前台可将VIP/急诊患者插入队首或指定位置 |
| 优先插队 | 优先级排序 | 队列按优先级自动排序：急诊>VIP>普通 |
| 牙椅排期 | 牙椅状态总览 | 展示所有牙椅当前状态（空闲/占用/维护） |
| 牙椅排期 | 时段排班表 | 按日期展示各牙椅的时段预约情况（甘特图风格） |
| 牙椅排期 | 预约操作 | 选择牙椅+时段进行预约，自动触发冲突校验 |
| 冲突校验 | 时段重叠检测 | 预约时检测所选时段是否与已有预约冲突 |
| 冲突校验 | 退订释放 | 患者取消预约后自动释放对应时段 |
| 冲突校验 | 复诊衔接 | 诊疗结束后可一键预约复诊，自动避开冲突时段 |

## 3. 核心流程

### 3.1 患者就诊流程
患者到店 → 前台取号（选择类型）→ 进入优先级队列 → 等待叫号 → 叫号上椅 → 诊疗中 → 诊疗完成 → 可选复诊预约

### 3.2 VIP/急诊插队流程
VIP/急诊患者取号 → 系统自动提升优先级 → 插入队列前方 → 优先叫号

### 3.3 预约冲突处理流程
选择牙椅+时段 → 冲突校验 → 无冲突则确认预约 / 有冲突则提示并推荐最近可用时段

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：医疗蓝(#0891B2) + 清新绿(#10B981) 代表专业与健康
- 辅助色：暖白(#F8FAFC)背景、灰色(#64748B)文字
- 强调色：急诊红(#EF4444)、VIP金(#F59E0B)
- 按钮风格：圆角胶囊按钮，轻微阴影，hover时上浮
- 字体：思源黑体(Noto Sans SC) + DM Sans 数字
- 布局：卡片式布局，底部Tab导航，移动优先
- 图标：Lucide线性图标

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 排队叫号 | 取号登记 | 表单卡片(姓名/手机/类型选择)、取号按钮、排队序号动画弹出 |
| 排队叫号 | 实时队列 | 列表卡片，每项含序号/姓名/类型标签(彩色)/等待时长，当前叫号高亮 |
| 排队叫号 | 叫号操作 | 大号叫号按钮(脉冲动画)、下一个按钮、跳过按钮 |
| 排队叫号 | 就诊状态 | 牙椅卡片网格，每张显示椅号/患者/开始时间/已用时长 |
| 优先插队 | VIP/急诊标识 | 彩色标签(金色VIP/红色急诊)，队列中醒目显示 |
| 优先插队 | 插队操作 | 拖拽排序或点击插队按钮，确认弹窗 |
| 牙椅排期 | 状态总览 | 牙椅卡片网格，绿色空闲/蓝色占用/灰色维护 |
| 牙椅排期 | 时段排班表 | 时间轴甘特图，色块表示预约占用，点击空白处新建预约 |
| 牙椅排期 | 预约操作 | 弹窗表单(选择患者/牙椅/日期/时段)，实时冲突提示 |
| 冲突校验 | 重叠检测 | 红色冲突警告条，显示冲突详情 |
| 冲突校验 | 退订释放 | 退订确认弹窗，释放后时段变绿动画 |
| 冲突校验 | 复诊衔接 | 复诊预约卡片，自动填充上次牙椅/医生，智能推荐时段 |

### 4.3 响应式设计
- 移动优先设计，主要面向手机端使用
- 卡片网格在小屏单列、大屏多列自适应
- 时段排班表支持横向滚动
- 底部Tab导航适配移动操作
